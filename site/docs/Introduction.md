 <h2> About </h2>
 
 MusicComp is a custom discord music bot with full Spotify, YouTube and Soundcloud API support. It also has a <a href="https://musiccomp.devcomp.tk/dashboard">Web Dashboard</a>.

 
## Installation
Invite the bot to your server at <a href="https://discord.com/oauth2/authorize?client_id=928617705983082527&permissions=2205281600&scope=bot%20identify%20guilds%20applications.commands&redirect_uri=https://musiccomp.devcomp.tk/api/callback&response_type=code">here</a>. 

**OR**

[Self Host](https://devcomp.tk/MusicComp-v2/docs/SELFHOST) the bot.



## Built with:
A lot of this bot relies on several other open source projects including: 
- [Lavalink](https://github.com/freyacodes/Lavalink)
